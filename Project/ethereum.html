What Is Ethereum?

Ethereum (ETH) is a decentralized, open-source blockchain system that features smart contract functionality. It is the second-largest cryptocurrency platform by market capitalization, after Bitcoin.

Ethereum was proposed by programmer Vitalik Buterin in late 2013 and development was crowdfunded in 2014. The network went live on July 30, 2015, with 72 million coins premined.

Ethereum allows developers to build and deploy decentralized applications (dApps), which are applications that run without any central control, making them resistant to censorship, fraud, and interference.
